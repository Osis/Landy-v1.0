CF Apps

Open Source iOS Client for Cloud Foundry

View and debug applications from your mobile devices on any Cloud Foundry installation.

Easily log into your favorite Cloud Foundry providers

Currently supports Atos, IBM Bluemix, Pivotal Cloud Foundry, Predix, or any other Cloud Foundry installation.

Check out your applications on the go

Have full access to application metadata, instance statistics, attached services and more!

Live application logging

Get access to live application logs and events from your mobile device. Fully supports split screen on iPads for live debugging.

Open Source

Source code is hosted on Github. Pull & Feature Requests are welcome from all.

Private & Secure

Serverless design means you communicate directly to Cloud Foundry. The application does not talk to any third part APIs or collect analytics.

Any Cloud Foundry

Connect to any hosted or private Cloud Foundry installation that your mobile device has access to.
